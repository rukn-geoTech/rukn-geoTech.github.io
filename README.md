# 基石智勘 Rukn GeoExplore — 官网

双语（中文 / English）单页官网。

## 文件说明
- **`index.html`** — 完整自包含网站，所有字体、图片、脚本已内联。**直接用于 GitHub Pages 即可**（无需其他文件）。
- **`source/`** — 可编辑源文件：
  - `Rukn GeoExplore.dc.html` — 页面源文件
  - `support.js` — 运行时
  - `logo_v1.png` / `logo_v1.svg` — 标志

## 部署到 GitHub Pages
1. 将本文件夹内容推送到仓库。
2. 仓库 Settings → Pages → Source 选择对应分支根目录。
3. 访问 `https://<用户名>.github.io/<仓库名>/` 即可看到 `index.html`。

右上角按钮可切换中文 / English。
